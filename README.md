# Responsive Homepage

This is my solution to the [Homepage project](https://www.theodinproject.com/lessons/node-path-advanced-html-and-css-homepage) from The Odin Project's Advanced HTML and CSS course. I built a responsive, portfolio-style homepage based on the supplied desktop, tablet, and mobile design references.

## About the project

The page presents a photographer's profile, a collection of project cards, and contact information. My main goal was to create a layout that remains balanced and readable across screen sizes from mobile phones to desktop displays.

I used CSS Grid for the overall page structure, Flexbox and Grid for the individual sections, and media queries to adapt the header, project gallery, footer, images, typography, and spacing at narrower viewport widths. JavaScript generates the project cards from a reusable data array.

## Features

- Responsive layouts for desktop, tablet, and mobile screens
- A header with an introductory profile and social links
- A responsive grid of dynamically generated project cards
- A contact footer with social links and a responsive portrait
- Expandable photo credits that keep attribution available without crowding the design
- Locally hosted Playfair Display and Roboto font files

## Built with

- Semantic HTML5
- CSS Grid and Flexbox
- CSS media queries
- Vanilla JavaScript
- [Devicon](https://devicon.dev/) social icons

## What I practiced

Through this project, I practiced translating separate design references into one fluid implementation. I also worked on responsive image sizing, flexible containers, reusable class naming, breakpoint-specific layout changes, and keeping content centered without relying on fixed viewport dimensions.

## Running locally

Clone or download the repository, then open `index.html` in a browser. No build step or package installation is required.
